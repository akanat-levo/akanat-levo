## The Short Version

I'm not a task-taker. Give me a problem and I'll come back with a shipped solution.

At **Levo.ai** I started as an intern and moved into a full-time Software Engineer role. I build the infrastructure customers depend on: REST APIs that dynamically toggle OWASP security rules on the API Firewall, reliability testing for the AI Gateway, and kernel-level traffic sensing with eBPF. I extended the nghttp2-based eBPF sensor to capture and process **HTTP/2 payloads up to 256 KB**, giving observability across high-volume network traffic.

Outside work I build to learn: a Django social platform end-to-end, and a Deep Q-Network agent that teaches itself to play Snake through experience replay.
